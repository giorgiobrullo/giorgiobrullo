## things I've built

**[enigmabin](https://github.com/giorgiobrullo/enigmabin)** · Zero-knowledge pastebin with end-to-end encryption. Classical or quantum-resistant.

**[kbye](https://kbye.lol)** · Encode secrets into "ok k bye lol cool nice sure yeah." For dry texters with something to hide.

**[house-always-wins](https://youlo.se)** · An interactive proof that gambling is a losing proposition. Simulations, math, and exposed mechanics.

**[doomer-clock](https://github.com/giorgiobrullo/doomer-clock)** · How much sand remains in your hourglass? A mortality visualizer.

**[cider-api](https://github.com/giorgiobrullo/cider-api)** · Async Rust client for the Cider music player. Published on [crates.io](https://crates.io/crates/cider-api).

**[cider-listen-together](https://github.com/giorgiobrullo/cider-listen-together)** · Listen to music together with friends using Cider. P2P sync via libp2p.

**[Europa](https://github.com/giorgiobrullo/Europa)** · A game built in Unity during my CS degree. Completed.
